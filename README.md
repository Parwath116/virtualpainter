Virtual Painter with Hand Gesture Control
Virtual Painter is a real-time drawing application that uses hand gestures to control painting actions through your webcam. Built with OpenCV and MediaPipe, it enables users to draw, erase, and mirror strokes without touching the screenâ€”just by moving their fingers.

Features
Hand Gesture Recognition:
1 Finger: Select color
2 Fingers: Draw
3 Fingers: Mirror draw
4 Fingers: Erase
Color Selection:
Red, Blue, Green, Yellow, and Blank (eraser)
Visual color palette displayed at the top of the screen
Face Detection:
Detects and highlights faces using Haar cascades
Canvas Overlay:
Persistent drawing canvas overlaid on live video feed
UI Feedback:
Displays current mode, selected color, and finger count
Real-time FPS counter
